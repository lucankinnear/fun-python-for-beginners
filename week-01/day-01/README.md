## Quick explanation
Python is a general-purpose programming language known for being readable and beginner-friendly.  
A Python program runs from top to bottom, executing each line in order.

To interact with users:
- `print()` displays output
- `input()` pauses the program and waits for user input

## Common beginner mistakes
- Expecting `input()` to return a number
- Forgetting to run the file after saving
- Panicking when an error appears instead of reading it

## Best practices
- Run small scripts often
- Read error messages carefully
- Keep programs simple while learning
